Git

Создание и клонирование

  • git init - создать новый репозиторий в текущей папке.
  • git clone "url" - скопировать удалённый репозиторий

    • Статус и история

  • git status - какие файлы изменены / добавлены / не отслеживаются
  • git diff - показать незакоммиченные изменения

    • Добавление в индекс

  • git add "файл" - добавить конкретный файл
  • git add . - добавить все изменения (новые, изменённые)
  • git add -A - добавить всё, включая удалённые файлы

    • Коммиты

  • git commit -m "Сообщение" - закоммитить подготовленные изменения
  • git commit -a -m "Сообщение" - автоматически добавить все отслеживаемые файлы и коммит

    • Работа с удалённым репозиторием

  • git push --set-upstream origin "сообщение" - первый пуш ветки main (установить upstream)
  • git push - отправить изменения на сервер
  • git pull - скачать изменения и сразу слить (fetch + merge)
  • git fetch - скачать изменения без слияния

    • Ветвление

  • git branch - список локальных веток
  • git branch "имя" - создать новую ветку
  • git checkout "ветка" - переключиться на ветку
  • git branch -d "ветка" - влить указанную ветку в текущую
  • git merge "ветка" - переключиться на ветку
